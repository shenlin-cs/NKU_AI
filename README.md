# NKU_AI
## 虚拟手术规划导航软件系统
基于C++、Qt和VTK独立开发一套虚拟手术规划导航软件系统并进行部署，完成手术规划导航方法设计，进行钻制通道的自主规划，实现术前演练。
## U-Net
对19组CT序列的每张CT图像进行耳蜗和面神经部分的标注，每组CT图像的有效数据量，即同时出现耳蜗和面神经的CT数据为8张至12张不等，利用标注得到的数据集训练一个识别耳蜗和面神经部分的U-Net网络，根据训练好的网络进行耳蜗和面神经部分的自动识别以进行后续的规划实验。
